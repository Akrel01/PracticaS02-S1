Etiquetas

Las etiquetas HTML son codigo para estructurar una pagina web.

Tipos de etiqueta:

Etiquetas de apertura y cierre: Las etiquetas de apertura indican donde empieza el elemento y la del cierre indica donde terminan las propiedadesd de la etiqueta de apertura

Self-closing: Son etiquetas que no necesitan ser cerradas.

En bloque e inline: Las etiquetas en bloque ocupan un espacio total en bloque dentro
del elemento padre y los elementos inline ocupan solo el espacio del contenido.

Atributos en Html:

Los atributos en etiquetas Html son palabras especiales que se utilizan dentro de la etiqueta de apertura de un elemento para controlar su comportamiento.
Los atributos proporcionan información adicional sobre el elemento

Encabezados en Html:

Los encabezados en Html son etiquetas de código que definen los títulos y subtítulos de una página web
